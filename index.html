Certificate trust

One-time setup
The installer now trusts the local CA for the current Windows user and configures PHP to use it for cURL/OpenSSL verification.
PowerShell
Import-Certificate -FilePath 'ssl\rootCA.pem' -CertStoreLocation 'Cert:\CurrentUser\Root'
Or manually via File Explorer:
  1. Open ssl\ in File Explorer
  2. Right-click rootCA.pem → Install Certificate
  3. Select Current User → Next
  4. Choose Place all certificates in the following store → Browse
  5. Select Trusted Root Certification Authorities → OK
  6. Next → Finish
  7. Fully close and reopen your browser
The generated server certificate covers localhost, 127.0.0.1, and the active machine IP from setup time.
